# 專案發展藍圖 (Roadmap)

## 近期優化目標 (Short-term Optimizations)

- [ ] **效能優化 (Performance)**
  - 圖片資源 WebP 轉換與壓縮。
  - Code Splitting：針對小遊戲組件進行 Lazy Loading，減少首屏載入時間。
- [ ] **SEO 增強**
  - 增加更動態的 Meta Tags。
  - 結構化資料 (JSON-LD) 標記。
- [ ] **PWA 支援**
  - 加入 `manifest.json` 與 Service Worker，讓網站可以被安裝在手機桌面。

## 新功能規劃 (New Features)

### 1. 互動性提升

- **Diesel Chatbot**: 訓練一個說話風格像 Diesel 的 AI 聊天機器人。
- **留言板 (Guestbook)**: 復古風格的留言板，讓訪客留下足跡。

### 2. 資料整合

- **Spotify 串接**: 顯示 Diesel 目前正在聽的 High歌。
- **Instagram Feed**: 自動抓取最新的 IG 貼文展示。

### 3. 進階小遊戲

- **Diesel Run**: 橫向捲軸跑酷遊戲。
- **打地鼠**: 只是地鼠換成了... (待定)。

## 維護計畫

- 每月檢查依賴套件 (Dependencies) 更新。
- 定期備份語錄與圖片資料。
