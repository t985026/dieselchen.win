# 使用說明手冊 (User Manual)

## 歡迎來到 DieselChen.win

這是一個充滿能量與 Diesel 風格的個人網站。以下是您可以在這裡體驗的功能：

### 1. 核心體驗

- **隨機名言**: 每次刷新或點擊，都能看到 Diesel 的不同面貌與至理名言。
- **視覺衝擊**: 請享受精心設計的各式轉場與視覺特效。

### 2. 特殊事件

- **休息提醒**: 如果您在特定時段瀏覽，可能會收到貼心的休息提醒。
- **下班慶祝**: 到了下班時間，網站會變身為慶祝模式，請打開音響享受這份喜悅！
  - *注意*: 音效可能會很震撼，辦公室瀏覽請注意音量。

### 3. 小遊戲中心 (Mini Games)

位於畫面左下角的 "G" 按鈕是您的舒壓中心：

- **拉霸機 (Slots)**: 試試您的手氣，看能不能轉出三個 Diesel！
- **什麼時候吃飯/喝什麼 (Roulette)**: 有選擇障礙？讓命運之輪幫您決定午餐或下午茶。
- **射飛鏢 (Darts)**: 簡單的舒壓小遊戲，點擊螢幕發射！

### 常見問題 (FAQ)

**Q: 為什麼會有聲音？**
A: 這是 Diesel 的熱情，您可以通過瀏覽器分頁靜音，但建議佩戴耳機體驗最佳效果。

**Q: 畫面跑版了怎麼辦？**
A: 請嘗試重新整理頁面，或確認您的瀏覽器是否為最新版本。本站支援響應式設計 (RWD)。
